# flomo-剪纸 浏览器插件

一个用于快速选取网页内容并保存到 flomo 的浏览器插件，支持标签管理和图片处理。

## 功能特性

- **内容选取**：精准选取网页中的文本和图片内容
- **全文获取**：一键获取当前网页的全部文本内容
- **标签管理**：添加自定义标签，支持标签持久化存储（12小时过期）
- **图片处理**：自动提取图片链接，最多支持9张图片
- **flomo 集成**：直接保存内容到 flomo，支持标签格式
- **用户友好**：直观的界面设计，流畅的操作体验

## 安装方法

### Chrome 浏览器
1. 打开 Chrome 浏览器，进入扩展管理页面（chrome://extensions/）
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目的 `browser_plugin` 目录
5. 扩展安装完成后，在浏览器右上角可以看到插件图标

### Edge 浏览器
1. 打开 Edge 浏览器，进入扩展管理页面（edge://extensions/）
2. 开启左侧的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目的 `browser_plugin` 目录
5. 扩展安装完成后，在浏览器右上角可以看到插件图标

## 使用方法

### 1. 配置 API Key
1. 点击插件图标，打开侧边栏
2. 点击右上角的菜单按钮（☰）
3. 选择「配置 flomo API」
4. 输入你的 flomo API Key（获取方式：登录 flomo -> 设置 -> API -> 复制 API Key）
5. 点击「保存」按钮

### 2. 选取内容
1. 在侧边栏中点击「选 取」按钮
2. 鼠标会变成十字形状，移动到你想要选取的内容上
3. 点击鼠标左键选取内容，被选中的内容会显示绿色边框
4. 按 Enter 键或点击顶部的「确认选取」按钮完成选取
5. 选取的内容会显示在侧边栏中

### 3. 获取全文
1. 在侧边栏中点击「全 文」按钮
2. 插件会自动获取当前网页的全部文本内容
3. 内容会显示在侧边栏中

### 4. 添加标签
1. 点击侧边栏底部的「添加标签」按钮
2. 在弹出的输入框中输入标签名称
3. 按 Enter 键添加标签
4. 标签会显示在内容区域上方
5. 最多支持添加10个标签

### 5. 保存到 flomo
1. 确认侧边栏中显示的内容无误
2. 点击底部的「保 存」按钮
3. 保存成功后会显示「保存成功」的通知
4. 内容会自动保存到你的 flomo 账号中

## 技术实现

- **前端框架**：原生 JavaScript + CSS
- **浏览器扩展 API**：Chrome Extension Manifest V3
- **存储方式**：localStorage（用于存储 API Key 和标签）
- **网络请求**：Fetch API（用于与 flomo API 通信）
- **内容选取**：DOM 操作 + 事件监听

## 注意事项

1. **API Key 安全**：API Key 仅存储在本地，不会上传到任何服务器
2. **图片限制**：最多支持处理9张图片，超过9张的图片将不会被处理
3. **标签过期**：标签会在12小时后过期，过期后需要重新添加
4. **内容安全**：某些网站可能会设置安全策略，阻止插件获取内容
